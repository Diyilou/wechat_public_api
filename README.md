# 简介

基于ruby的微信公众号开发的API，包含对微信公众号菜单栏、客服消息、模板消息、帐号管理等接口的封装。

API长期更新维护，建议使用最新版本。

## 安装

添加 Gemfile:

```ruby
gem 'wechat_public_api'
```

执行 bundle:

    $ bundle install

或者直接通过 gem 安装:

    $ gem install wechat_public_api
